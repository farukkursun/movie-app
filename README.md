# React Movie App

Responsive Movie App made using React and Bootstrap.

<hr />

## What I Used

- Firebase
- React Router Dom
- React Toastify
- React-Bootstrap
- Context


### Installation Options

```
$ git clone https://github.com/farukkursun/movie-app
```

```
$ npm install
```

```
$ npm start
```

### Or

```
$ git clone https://github.com/farukkursun/movie-app
```

```
$ yarn
```

```
$ yarn start
```
